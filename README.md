## Kurulum (5 adım, 10 dakika)

### 1. Node.js yükleyin
https://nodejs.org adresine gidin, "LTS" sürümünü indirip kurun.

### 2. Bu klasörü bir yere koyun
Masaüstünüze ya da istediğiniz bir yere kopyalayın.

### 3. Terminali açın, klasöre gelin
```
cd masaustu/ink_website
npm install
```

### 4. GitHub'a yükleyin (ücretsiz)
- github.com adresine gidin, hesap oluşturun
- "New repository" > "ink-website" adıyla oluşturun
- Tüm dosyaları yükleyin

### 5. Vercel'e bağlayın (ücretsiz hosting)
- vercel.com adresine gidin
- "Import Git Repository" > GitHub hesabınızı bağlayın
- ink-website reposunu seçin > Deploy

### Domain bağlama
- Vercel dashboard > Settings > Domains
- ilahinizamvekainat.tr ve ilahinizamvekainat.info ekleyin
- DNS ayarlarını domain sağlayıcınızdan yapın (Vercel size söyler)
